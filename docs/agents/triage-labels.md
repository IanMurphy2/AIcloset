# Triage Labels

Los skills usan cinco roles canónicos de triage. Esta tabla mapea esos roles a los strings reales usados en Linear.

| Label en mattpocock/skills | Label en Linear      | Significado                                          |
| -------------------------- | -------------------- | ---------------------------------------------------- |
| `needs-triage`             | `needs-triage`       | Un maintainer necesita evaluar este issue            |
| `needs-info`               | `needs-info`         | Esperando más información del reportador             |
| `ready-for-agent`          | `ready-for-agent`    | Completamente especificado, listo para un agente AFK |
| `ready-for-human`          | `ready-for-human`    | Requiere implementación humana                       |
| `wontfix`                  | `wontfix`            | No se va a accionar                                  |

Cuando un skill mencione un rol (ej. "aplica el label AFK-ready"), usa el string de la columna derecha.

Edita la columna derecha si en algún momento cambias el vocabulario de labels en Linear.
